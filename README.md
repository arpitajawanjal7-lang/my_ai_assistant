MediCare Hospital Assistant Bot

## Project Overview
MediCare Hospital Assistant Bot is an AI-powered healthcare chatbot designed to assist patients with hospital-related information through an interactive conversational interface.

## Features
- Hospital Information Assistance
- Department Navigation
- Appointment Guidance
- Patient Support Services
- AI-Powered Chatbot
- User-Friendly Interface

## Technology Stack
- Botpress Cloud
- Generative AI
- Natural Language Processing (NLP)
- Web Chat Interface

## System Workflow
1. User opens the chatbot.
2. User selects a department.
3. Chatbot understands the query.
4. Chatbot provides relevant information.
5. User receives instant assistance.

- Multilingual Support
